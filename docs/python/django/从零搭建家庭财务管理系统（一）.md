## 一、简介
从大学毕业到开始工作，接触Python也有几年的时间了，到现在（2022）自己居然无法写出一个完整的项目来。之前的学习内容、路线，包括学习方法，感觉都有些问题，借着疫情在家办公的机会，把每天节省下来的通勤时间继续用在学习上，争取写出一个完整的项目，也给自己的python学习交上一份满意的答卷。
